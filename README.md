# Perfect Nkosi - Portfolio
## ***Feel free to use, and change the code. Go nuts! 😎***

simply 
``` 
    run 'yarn' to install dependencies
    run 'yarn dev' to view the website
```

## Tools used:
- React Js
- Tailwindcss
- Sanity for content management
- Vite as the bundler
- Yarn for managing packages
